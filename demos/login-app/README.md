# Login Application with FastAPI and Docker

A complete login system with:
- FastAPI backend (Python)
- Vanilla JS frontend
- SQLite database
- Dockerized deployment

## Features
- User registration
- Secure password hashing (bcrypt)
- JWT authentication
- Protected routes
- Responsive UI

## Prerequisites
- Docker
- Docker Compose

## Setup
1. Clone the repository
2. Build and start containers:
   ```bash
   docker-compose up --build
